== Description ==

Offer free gifts to your customers based on cart total amount.

This WooCommerce plugin enables you to offer free gifts to your customers on checkout.

* Set minimum cart amount 
* Set maximum cart amount 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Woocommerce->Setting->Gift Name screen to add gift
